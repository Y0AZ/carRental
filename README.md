## Aymeric YAPI | M1 EISI-SIN

## Sujet

Application de gestion d'une agence de location de véhicules. Le système permet de gérer les véhicules, les clients et les réservations avec des règles métier complètes : tarification dynamique selon la saison et la catégorie du véhicule, gestion de la caution, facturation des dépassements kilométriques, et gestion des indisponibilités pour révision.
